Skin Diseases Detection

This page provides an introduction to the skin diseases detection model.

Model Introduction

This skin diseases detection model is fine-tuned from the state-of-the-art Facebook/ConvNeXT-Base model available on Hugging Face.

The HAM10000 Dataset

The model is trained on the "HAM10000" dataset, which is a collection of dermatoscopic images used for training neural networks to diagnose pigmented skin lesions automatically. This dataset is highly valuable for researchers and practitioners in both the dermatology and machine learning fields.

The dataset addresses the challenges of small dataset sizes and lack of diversity by offering a wide range of images collected from different populations using various imaging modalities. It comprises a total of 10,015 dermatoscopic images covering various diagnostic categories associated with pigmented skin lesions:

  • Actinic keratoses and intraepithelial carcinoma (akiec)
  • Basal cell carcinoma (bcc)
  • Benign keratosis-like lesions (solar lentigines, seborrheic keratoses, and lichen-planus like keratoses - bkl)
  • Dermatofibroma (df)
  • Melanoma (mel)
  • Melanocytic nevi (nv)
  • Vascular lesions (angiomas, angiokeratomas, pyogenic granulomas, and hemorrhage - vasc)

For reliability, over 50% of the lesions were confirmed through histopathology. For the remaining cases, ground truth was established through follow-up examinations, expert consensus, or confirmation using in-vivo confocal microscopy. The dataset also addresses the need for multiple images of the same lesion using the `lesion_id` column in the "HAM10000_metadata" file for tracking.

Due to upload size limitations, the dataset is divided into two files:

  1. "HAM10000_images_part1.zip" (containing 5,000 JPEG files)
  2. "HAM10000_images_part2.zip" (containing 5,015 JPEG files)

Researchers can utilize this dataset to develop and evaluate AI-based systems for classifying and diagnosing skin lesions. It offers a diverse and representative set of images, making it a valuable resource for advancing automated diagnosis through machine learning algorithms.
