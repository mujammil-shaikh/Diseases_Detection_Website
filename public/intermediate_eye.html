Eye Diseases Detection

This page provides an introduction to the eye diseases detection model.

Model Introduction

This eye diseases detection model is fine-tuned from the state-of-the-art Facebook/ConvNeXT-Base model available on Hugging Face.

Ocular Disease Intelligent Recognition (ODIR) Dataset

The model is trained on the "Ocular Disease Intelligent Recognition (ODIR)" dataset, which contains structured ophthalmic data from 5,000 patients. The dataset includes patient age, color fundus photographs for both left and right eyes, and diagnostic keywords provided by doctors. The data was collected from various hospitals and medical centers in China by Shanggong Medical Technology Co., Ltd. Different cameras were used to capture fundus images, leading to varying image resolutions. Trained human readers annotated the data with diagnostic labels, resulting in eight possible labels:

  • Normal (N)
  • Diabetes (D)
  • Glaucoma (G)
  • Cataract (C)
  • Age-related Macular Degeneration (A)
  • Hypertension (H)
  • Pathological Myopia (M)
  • Other diseases/abnormalities (O)
